## CBSE Syllabus ChatOps
A CBSE course summarizer with a chat bot interface

<br />

### Features

- [X] Course summarizer
- [X] Chat interface
- [X] Support for different languages
- [X] Granular commands, such as reading a particular chapter
- [X] Web preview
- [X] Course fetch and seed summarizer
